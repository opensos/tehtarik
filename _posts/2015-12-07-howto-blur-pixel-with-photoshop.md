![Ahnaf](https://pbs.twimg.com/profile_images/671975611309514753/DuWMSyGZ_400x400.png)

1. Open Photoshop
2. Browse For Picture 
3. Select Using Marquee Tool
4. Goto Menu Filter > Pixelate > Mosaic
5. Choose Cell Size Then Click Ok
6. Save picture
